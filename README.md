A React single-page application that replicates parts of the Stripe marketing site (navbar, hero, support section, and more).

## Prerequisites

- Node.js (LTS version recommended) installed on your machine.
- npm (comes with Node) or Yarn installed globally.

You can check versions with:

node -v
npm -v

## How to run this project

1. Download the project ZIP file.
2. Extract the ZIP file.
3. Open a terminal and go into the folder:

cd stripe-copy

4. Install dependencies:

npm install

5. Start the development server:

npm start

6. Open your browser and go to:

http://localhost:3000
